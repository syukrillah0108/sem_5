Detail Kursus: Belajar Python untuk Pemula

Deskripsi Kursus

Kursus ini dirancang untuk pemula yang ingin mempelajari dasar-dasar pemrograman Python. Anda akan mempelajari konsep dasar seperti variabel, tipe data, loop, fungsi, dan banyak lagi. Kursus ini dilengkapi dengan video pembelajaran, kuis interaktif, dan proyek akhir.

Yang Akan Anda Pelajari

  • Pengenalan Python
  • Struktur Data (List, Tuple, Dictionary)
  • Kontrol Alur Program (If-Else, Loop)
  • Fungsi dan Modul
  • Proyek Akhir: Membuat Program Sederhana

Durasi Kursus

10 Jam (Video dan Praktek)
